# アプリケーションの概要
通知機能のある階層型ToDoアプリです。
親階層(カテゴリー)にスケジュール登録できるため、スケジュールは一定のまま子階層(タスク)のみ変えることができます。
単純なToDoアプリとしてだけではなくルーティンを生活に取り入れたい方、アイディアメモなどとしても使えるアプリとなっております。

# アプリケーションの機能一覧
- カテゴリー・タスク登録
- カテゴリー・タスク編集
- カテゴリー・タスク削除
- スケジュール登録
- 保存機能
- 通知機能
- タスク一括削除
